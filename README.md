# Credit Scoring Model Using Machine Learning

## Project Overview
This project predicts customer creditworthiness using Machine Learning classification algorithms.

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Data Preprocessing
- Feature Engineering
- Data Visualization
- Model Evaluation
- ROC Curve Analysis
- Feature Importance Analysis

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Outcome
Random Forest Classifier achieved the best performance for predicting credit risk.
